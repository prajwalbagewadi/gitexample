Animation in Css

  • @keyframes: specifies animation-code
  • animation: set the animation property
  • animation-delay: delay at the start of animation
  • animation-direction: animation should play forward or backward or alternate cycles
  • animation-duration: specifies the time it takes to complete one animation cycle
  • animation-iteration-count: specifies number of times animation should be played
  • animation-name: name of @keyframes animation
